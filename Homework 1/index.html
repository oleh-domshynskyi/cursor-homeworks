Основні команди Git

  • git config --global user.name - вказати ім’я користувача
  • git config --global user.email - вказати пошту користувача
  • git init - команди для створення Git репозиторію
  • git status - показує стан репозиторію
  • git add - індексує файл для наступного коміта
  • git commit - закомітити зміни у файлах
  • git log - перегляд історії комітів
  • git checkout - можна пересуватись по комітах та гілках
  • git branch - створення нової гілки
  • git push - завантажує локальні зміни у віддаллений репозиторій
  • git pull - завантажує зміни з відддаленого репозиторію у локальний
  • git merge - зливає гілки в одну
  • git clone - копіює репозиторій
  • .gitignore файл - Git ігнорує перераховані файли
  • git revert - відміняє зміни в коміті
  • git reset - повністю видаляє зміни
  • git rebase - перезаптсує історію всіх комітів
  • git cherry-pick - перенесення комітів між гілками
  • git stash - тимчасово зберігає незафіксовані зміни
